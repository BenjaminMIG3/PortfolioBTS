Sudoku : Jeu en C avec curses pour le terminal

Ce projet est une implémentation du jeu de Sudoku, où le joueur peut résoudre une grille 9x9 directement dans l’invite de commande. Développé en C avec la bibliothèque `curses`, il utilise une interface textuelle pour afficher la grille et permettre à l’utilisateur d’interagir en saisissant des chiffres, avec une validation des règles du Sudoku.

  • Utilisation de curses pour une interface textuelle dynamique dans le terminal.
  • Programmation en C pour la logique du jeu et la validation des entrées.
  • Affichage et interaction dans l’invite de commande.

Ce projet met en avant mes compétences en programmation en C, ainsi que ma capacité à utiliser la bibliothèque `curses` pour créer des applications interactives en mode texte, adaptées à un environnement terminal.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran du jeu est disponible ci-dessus.
