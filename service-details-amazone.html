Àmazone : Application desktop avec C#, Avalonia et .NET

Àmazone est une application desktop multiplateforme développée en C# en utilisant le framework Avalonia et .NET. Inspirée par les plateformes e-commerce, elle offre une interface moderne et réactive permettant aux utilisateurs de gérer des produits ou des données de manière intuitive, peu importe leur système d’exploitation.

  • Utilisation d’Avalonia pour une UI multiplateforme.
  • Développement en C# avec .NET pour des performances optimales.
  • Conception d’une interface utilisateur fluide et moderne.

Ce projet illustre mes compétences en développement d’applications desktop avec C#, ainsi que ma capacité à utiliser des frameworks modernes comme Avalonia pour créer des solutions portables et esthétiques.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran de l’application est disponible ci-dessus.
