HollyFork : Application web avec Python, Django et hébergement Ubuntu

HollyFork est une application web conçue avec Python et le framework Django, offrant une solution robuste pour la gestion de données ou de services. Hébergée sur un serveur Ubuntu, elle garantit une accessibilité continue et une administration simplifiée grâce à un environnement Linux stable.

  • Développement backend avec Django pour une structure performante.
  • Hébergement sur un serveur Ubuntu avec configuration personnalisée.
  • Interface utilisateur responsive intégrée avec Bootstrap.

Ce projet met en avant mes compétences en développement web avec Python et Django, ainsi que ma capacité à déployer et gérer une application sur un serveur Linux comme Ubuntu.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran de l’application est disponible ci-dessus.
