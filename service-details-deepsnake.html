DeepSnake : Un Snake moderne avec Python et Pygame

Ce projet est une implémentation du jeu classique Snake, où le joueur contrôle un serpent qui grandit en mangeant des pommes tout en évitant les collisions. Développé avec Python et la bibliothèque Pygame, il démontre mes compétences en logique de programmation et en gestion d'événements graphiques.

  • Utilisation de Pygame pour la création d'une interface graphique.
  • Gestion des entrées utilisateur et de la boucle de jeu.
  • Code modulaire et réutilisable basé sur la programmation orientée objet.

Le projet a été une excellente opportunité pour approfondir mes connaissances en Python tout en créant une expérience interactive et amusante.

Vous pouvez voir une démonstration en image ci-dessus ou consulter le code source sur mon GitHub pour plus de détails.
