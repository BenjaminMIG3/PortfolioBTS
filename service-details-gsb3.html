GSB3 : Site web pour conférences médicales avec PHP et CodeIgniter

GSB3 est un site web développé en PHP avec le framework CodeIgniter 4, dédié à la gestion de conférences sur le thème médical. Il permet aux utilisateurs de s’inscrire, de réserver des billets pour les événements, de générer des billets numériques et de laisser des avis via un système de likes, offrant une expérience interactive et fonctionnelle.

  • Inscription et réservation de billets avec génération automatique.
  • Système d’avis basé sur des likes pour évaluer les conférences.
  • Backend robuste avec CodeIgniter 4 pour une gestion efficace.

Ce projet met en avant mes compétences en développement web avec PHP, ainsi que ma maîtrise de CodeIgniter pour créer des applications dynamiques et sécurisées, adaptées à un contexte médical.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran du site est disponible ci-dessus.
