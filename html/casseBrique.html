Projet casse-briques en Java

Ce projet a été produit durant ma premère année de licence mathématiques-informatiques dans le cadre d'un projet de fin de semestre.

Ce projet est codé en Java sous l'IDE processing il consiste en un jeu de casse-briques en 2D avec une raquette pilotable avec la souris est une balle qui rebondit sur la raquette et détruit une brique si il la touche

Le but est de détruire toutes les briques en les touchants avec la balle

La belle se depace graçe à l'appel de cette fonction dans la fonction miseAJourBalle

J'ai implémenté un menu pause

Lorsque toute les briques sont détruites nous devons touchés 10 fois la cage du boss afin de le délivrer ensuite la partie est terminée

Si la balle tombe 3 fois on perd la partie est un ecran game over s'affiche
