Application web de reservation de conférences en PHP avec le framework CodeIgniter4

Ce projet est dans la continuité des projets GSB.

Lorsque l'on entre dans le site une page de connexion apparait

Une fois connecter une page d'acceuil se présente à nous avec une navbar

Nous pouvons cliquer sur Consulter les conférences afin d'afficher les conférences stockées dans la BDD

Afin de consulter les presentations de cette conférence nous pouvons cliquez sur le bouton + de la conférence de notre choix

Suite à cela une page avec les présentation de la conférence s'affiche

Si la date de la présentation est passée on ne peut pas reserver un siege sinon on clique a nouveau sur le bouton plus de notre choix

Cela nous emmene sur la page de reservation d'un siège

Lorsque l'on clique sur un siège cela le reserve et le passe donc en rouge

Si un siège est deja occupé il est grisé donc on ne peut pas le reserver

Une fois avoir réserver un siège on apercois dans l'onglet mes reservations une ligne qui résume notre reservation

Si la date de la reservation est passée on peut la liker ou non

SI l'on clique sur le bouton telecharger cela nous genere un pdf du ticket de reservation
