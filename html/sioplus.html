SIO+ JAVA: Conception et Développement d'une Application java de Gestion de programmes et utilisateurs

Tout d'abord une page de connexion se présente lors du lancement de l'application

Si l'on ne détient pas de compte on peut créer un compte via un formulaire

Une fois connecté ont peut accéder et rechercher les utilisateurs recensés dans la base de données

Lorsque l'on clique sur un des utilisateurs ses logs de connexion apparaissent avec ses caracteristiques

On a la possibilité de consulter les programmes présents dans la base de données avec pour chaque programme sa description

Nous avons aussi la possibilité de créer des programmes via un formulaire

Un programme a un genre que l'on peut aussi créer
