GSB2 : Gestion de création de conférences avec Java POO

GSB2 est une application desktop développée en Java avec une programmation orientée objet (POO). Elle offre une solution pour gérer des conférences, avec des fonctionnalités telles que la création de nouvelles conférences, l’affichage des événements existants, la recherche dans le catalogue et la suppression d’entrées, le tout dans une interface simple et fonctionnelle.

  • Conception POO pour une structure modulaire et maintenable.
  • Fonctionnalités de gestion : création, affichage, recherche, suppression.
  • Gestion locale des données pour une organisation efficace.

Ce projet met en avant mes compétences en développement Java et en programmation orientée objet, avec une focus sur la création d’outils de gestion pratiques pour des événements comme des conférences.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran de l’application est disponible ci-dessus.
