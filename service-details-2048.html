2048 : Remake en C pour le terminal

Ce projet est une implémentation du jeu 2048, où le joueur combine des tuiles numérotées sur une grille 4x4 pour atteindre la tuile 2048. Développé en C, il utilise une interface textuelle dans l’invite de commande, permettant des déplacements via les touches directionnelles ou des commandes spécifiques, avec une logique de fusion et de score.

  • Interface textuelle simple dans l’invite de commande.
  • Programmation en C pour la logique de jeu (fusion, déplacements).
  • Gestion des entrées utilisateur et affichage en temps réel.

Ce projet met en avant mes compétences en programmation en C, avec une focus sur la création d’une application interactive en mode texte, adaptée à un environnement terminal.

Pour plus de détails ou pour consulter le code source, visitez mon GitHub. Une capture d’écran du jeu est disponible ci-dessus.
